# Quote API - NestJS

Eine einfache REST API für inspirierende Zitate, gebaut mit NestJS.

## Beschreibung

Diese API stellt eine Sammlung von inspirierenden Zitaten berühmter Persönlichkeiten bereit. Das Projekt dient als Beispiel für eine NestJS-Anwendung mit Repository-Pattern.

## Installation

```bash
# Abhängigkeiten installieren
npm install
```

## Anwendung starten

```bash
# Entwicklungsmodus
npm run start

# Watch-Modus (automatisches Neuladen)
npm run start:dev

```

## Technologien

- Nestjs CLI
- Typescript

**Hinweis**
Dieses Projekt ist für Bildungszwecke erstellt.
